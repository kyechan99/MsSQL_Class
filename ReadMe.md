# MsSQL_Class
 MsSQL 수업때 내준 과제 프로젝트를 올리고 정리하여 개선해서 올립니다.
 
## 사용시 유의 해야 할 점
### php
php 파일은 htdocs/DB_Class/ 폴더 경로 안에 모두 넣습니다.

### unity
unity 의 scene 은 LoginScene 부터 시작하여야 합니다.

### procedure
파일명 대로 프로시저가 있어야 합니다.
파일명 대로만 프로시저 생성하시고
프로시저 수정에서 txt 파일 안 내용으로 교체해 주시면 됩니다.
> 교체시 procedure 파일 맨 위 USE [] 에 데이터베이스 명으로 교체합니다.
